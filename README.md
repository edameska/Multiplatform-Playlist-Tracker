# Multiplatform Playlist Tracker

A web application to track, and analyze playlists across multiple platforms, built with **Yii2** and **PostgreSQL**.

## Features

- Connect and store playlists from multiple music platforms (Spotify, Youtube, etc.)
- Track song metadata (title, artist, duration, album)
- Store and manage user accounts securely

## Technologies Used

- **Backend:** Yii2 PHP Framework
- **Database:** PostgreSQL
- **Frontend:** Yii2 Views
- **Authentication:** Secure password storage with hashed passwords
- **APIs:** Spotify Web API, Youtube API
